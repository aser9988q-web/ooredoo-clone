# Ooredoo Kuwait Payment System

This is the complete Ooredoo Kuwait payment system website with all pages, styles, and assets.

## Pages Included

- `index.html` - Home page with payment form
- `knetpage.html` - KNET payment gateway
- `knetwait.html` - Loading page
- `knet/ver.html` - OTP verification
- `cvv.html` - CVV verification
- `hawety.html` - Hawety/ID verification
- `dashboard.html` - Admin dashboard
- `login.html` - Login page

## Assets

- Images and icons in `asderw-link.com/` directory
- CSS stylesheets in `knet/` directory
- Font files in `fonts.googleapis.com/` and `fonts.gstatic.com/`

## Installation

1. Clone the repository
2. Open any HTML file in your browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   ```
